Сайт бронирования
=================

## Описание

- Резервирование брони в выбранном ресторане для выбранного клиента. В ресторанах места
ограничены, поэтому свободное время будет зависеть от свободных мест.


## Примеры запросов

- example POST orders/create
```json
{
    "useruuid" : "56a5d05d-d43c-4c6d-8a01-f1602ef2f656",
    "restaranuuid" : "3a13632d-2177-47e8-9995-fd552ae4f717",
    "starttime" : "2025-04-22T15:00:00.123Z",
    "endtime" : "2025-04-22T17:00:00.123Z"
}
```
